Setting Up Your Environment

Before any great programmer can begin programming, it is necessary for everything to be set up correctly. In the world of FIRST Robotics, this has to be done a very specific way. Luckily for us, there is extensive documentation on this entire process, so it does not need to be fully described here. If you have not yet used the WPI Screen Steps Live, I would suggest giving it a quick read because I will be referencing it frequently in this section so that you can have detailed instructions for each step of this important process.
