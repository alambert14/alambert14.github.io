Installing Java

In order to program in Java, it is super important that the Java JDK (Java Developer's Kit) is installed on your computer. One advantage of Java is that it is not platform specific, meaning you can use it with any type of computer; however, that means that you must download the specific version for your operating system. In a nutshell, it just needs to be downloaded from Oracle and then linked in Eclipse, which will be described in the next section.

However, Java also needs to be installed to the roboRIO (the computer on the robot). This is so that the Java code can be compiled on the robot itself. WPI Screen Steps has steps for this as well, just follow them exactly.
