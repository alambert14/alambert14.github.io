Using Eclipse

Eclipse is an incredibly common Integrated Development Environment (IDE) for Java. It includes many features, but we only use a couple for programming the robot. One of the ways in which Eclipse is very useful is its modular capabilities; the user can have multiple windows open with different properties. Here are some important ones that you need to know:

Editing Window

This is the main window in which all of the code is written. Each file recently opened will appear as a tab, and the code will appear below to be edited.

Project Manager

This is the window that shows all of the projects in the workspace and the packages and files they contain. This is how you can access each file you want to work on and organize them in a graphic interface.

GIT

There are multiple windows that facilitate the usage of Git with Eclipse. To learn more about Git, jump to this section of the guide.

Console

Just like any compiler, Eclipse includes a console that alerts the programmer of any errors in the code and whether or not the program was able to build.

RIO Log

This module is included in the FRC Plugins, and is a console output for the roboRIO. This can be used to see text outputs from the code; it will NOT appear in the regular console.

Most of the personal settings that you might decide to alter are under Window > Preferences.
