Programming in Java

Now it is time for the chapter we have all been waiting for; how to actually program all of this junk we've been talking about! Obviously, without code, the robot would be less of a robot and more of a hunk of metal or wood (or whatever floats your boat). Many of these methods are only suggestions, and as mentioned later, this guide will only discuss one model of FRC programming. Feel free to make your code as simplistic or complicated as you would like.

DISCLAIMER: This part of the guide assumes that you have basic Java experience. There is not enough time or space to go over how to start writing Java from scratch. Start by pairing up with someone who is more experienced for your first time and learn as you go. If you have the chance, I would recommend the Java tutorials on Codecademy or taking AP Computer Science A if it's available to you.
